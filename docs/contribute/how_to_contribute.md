# How to Contribute

Melusine welcomes contribution from every volunteer or developer that makes use of it.

- If you wish to contribute, please read the [`CONTRIBUTING.md`](https://github.com/MAIF/melusine/blob/master/CONTRIBUTING.md) (external link) file you will find in the code repository.
- If you found an issue or wish to request a new feature for Melusine, please [file an issue on the issue tracker](https://github.com/MAIF/melusine/issues) (external link).
